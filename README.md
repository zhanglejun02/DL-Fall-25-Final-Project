# DL Final Project


### Install optimaizer frome souce

```bash
git clone https://github.com/zhanglejun02/DL-Fall-25-Final-Project.git
pip install -e .
```

### Install experiment dependencies

```bash
pip install -r exp_requirements.txt
```


## Usage

More details can be found in torchrun_main.py.
